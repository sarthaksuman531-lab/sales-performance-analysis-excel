## Sales Performance Analysis (Excel)

### Objective
Analyze sales data to understand performance across regions, categories, and time periods.

### Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

### Key Insights
- West and East regions generate higher sales compared to other regions
- Technology category contributes the highest revenue
- Sales show fluctuations over time, indicating varying demand patterns

### Outcome
Built a clean Excel dashboard to support data-driven business insights.
